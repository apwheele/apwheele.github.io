Overview

These are my tutorials for my crime analysis course. They are mostly in Excel, although a few show how to use SQL in Access, and one shows how to make BOLOs in Publisher. Materials are all posted online to run through the tutorials. If links don't work, send me an email (apwheele@gmail.com). Feel free to use as you wish (if an instructor feel free to email for example final product I used for class).

For a rough outline of the weeks:

  • Lab 01: (Excel) Intro to Excel & making a graph
  • Lab 02: (Excel) Pivot Tables in Excel
  • Lab 03: (Excel) Time Series Analysis
  • Lab 04: (Excel) Mapping Facilities in Excel (needs Windows to use many of these)
  • Lab 05: (Excel & Near-Repeat Tool) Aoristic analysis and near-repeat analysis
  • Lab 06: (Excel) Weighted Displacement Difference test
  • Lab 07: (Publisher) Creating Bulletin in Publisher
  • Lab 08: (Access) Using SQL to create Social Network
  • Lab 09: (Access) Using SQL to create chronic offender scores
  • Lab 10: Empty (so far!)
  • Lab 11: (Excel) Creating an officer dashboard using conditional highlighting, sparklines, making a nice print layout. Also makes a funnel chart.

For Lab 12 I had students do their own analysis of the Texas Deaths in Custody dataset. I have a prepared Excel file here, which also includes offender counts in each facility.
